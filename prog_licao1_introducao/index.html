1. O que é Programação?

Programar é simplesmente: Dar instruções ao computador para ele fazer algo

Exemplo simples:

Quando você abre o WhatsApp

Quando faz um pagamento

Quando vê um vídeo

👉 Tudo isso existe porque alguém programou.

🤖

2. O que é um Programa?

Um programa é um conjunto de instruções.

Exemplo na vida real:

  1. 1 Acordar
  2. 2 Escovar os dentes
  3. 3 Comer
  4. 4 Ir trabalhar

👉 Isso é um "programa humano" 😄

No computador:

É igual, mas usando código!

programa.py
print("Bom dia!")
print("Vamos programar")
💻

6. Lógica (O Mais Importante!)

Antes de programar, você precisa aprender a pensar logicamente.

Exemplo do dia a dia:

👉 "Se chover, fico em casa.

Se não chover, saio."

Tomada de decisão

Em programação:

if chuva:
    ficar_em_casa()
else:
    sair()

Isso é um "condicional"
