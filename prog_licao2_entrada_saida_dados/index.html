1. Mostrar mensagem com print()

O comando print() serve para mostrar algo na tela.

Código:

exemplo.py
print("Assalamu alaikum")

Saída:

Assalamu alaikum

O texto aparece exatamente como foi escrito.

🧠

2. Receber dados com input()

Agora vamos permitir que o usuário responda:

input_exemplo.py
nome = input("Qual é o seu nome? ")

O que acontece:

1 O programa mostra a pergunta na tela
2 O usuário digita a resposta
3 A resposta é guardada na variável nome
🔗

3. Juntando Tudo

Programa Completo:

programa.py
# Pergunta o nome
nome = input("Qual é o seu nome? ")

# Mostra a resposta
print("Prazer em te conhecer, " + nome)

Exemplo de execução:

Qual é o seu nome? Ali
Prazer em te conhecer, Ali

O símbolo + junta (concatena) texto com variável!

5. Erros Comuns (Cuidado!)

❌ ERRO 1: Esquecer o +

❌ Código Errado
print("Olá " nome)

✔️ Correto:

print("Olá " + nome)

Sempre use o + para juntar texto com variável!

❌ ERRO 2: Juntar texto com número

❌ Código Errado
print("Idade: " + 20)

✔️ Correto:

print("Idade: " + str(20))

Ou use input() que já retorna texto:

idade = input("Idade: ")
print("Idade: " + idade)

Dica: input() sempre retorna texto, então pode usar + direto!

❌ ERRO 3: Aspas não fechadas

❌ Código Errado
print("Olá Mundo)

✔️ Correto:

print("Olá Mundo")

Sempre feche as aspas! "texto" ou 'texto'
