Parte 1: Fundamentos

O que são Variáveis?

Variáveis são como "caixas" na memória do computador. Cada caixa tem um nome (etiqueta) e um valor (conteúdo).

Analogia Simples

Imagine uma variável como uma gaveta etiquetada:

  • A etiqueta é o nome da variável
  • O conteúdo da gaveta é o valor
  • Você pode olhar o conteúdo ou trocá-lo
let nome = "João"
👤
Variável "nome" contém "João"

Os 3 Tipos de Dados Principais

String (Texto)

Representa texto - qualquer sequência de caracteres entre aspas.

Exemplos

"FB Online" "Olá, Mundo!" "idade = 25"

Quando Usar

  • Nomes de usuários
  • Mensagens
  • Endereços
  • Emails
Parte 2: Prática

Declarando no Código

Veja como usar const e let na prática!

Dicas Importantes
  • Use const por padrão
  • Mude para let se precisar atualizar
  • Nomes devem ser descritivos
exemplo.js
// Exemplo com const
const empresa = "FB Aulas";
const anoFundacao = 2020;
const ativa = true;
 
// Tentar mudar causa erro!
// empresa = "Outra"; ❌
O que você aprendeu:
  • 1 const declara uma constante
  • 2 O valor não pode ser alterado depois
  • 3 Tentar reatribuir causa erro
