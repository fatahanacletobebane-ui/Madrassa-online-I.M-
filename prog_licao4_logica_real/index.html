🔀

Estruturas Condicionais

Aprenda a tomar decisões no código usando if, else if e else. Compare valores e execute diferentes ações baseadas em condições.

if (idade >= 18) { console.log("Maior de idade"); } else { console.log("Menor de idade"); }
🔁

Operadores Lógicos

Combine múltiplas condições usando AND (&&), OR (||) e NOT (!). Crie verificações complexas para validar dados.

if (saldo > 0 && status === "ativo") { permitirTransacao(); }
📊

Comparações

Utilize operadores de comparação para verificar igualdade, diferença, maior/menor que. Essenciais para filtros e validações.

// Operadores: ==, ===, !=, !==, >, <, >=, <= const aprovado = nota >= 10 && faltas < 5;
🎯

Switch Case

Quando há múltiplas condições, use switch para organizar melhor o código. Ideal para menus e estados.

switch(opcao) { case 1: depositar(); break; case 2: levantar(); break; default: erro(); }
