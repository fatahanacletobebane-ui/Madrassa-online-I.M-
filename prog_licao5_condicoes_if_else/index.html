Condições IF e ELSE

O que é? IF Simples IF e ELSE Exemplos Praticar

🤔 O que são Condições?

Imagine que você está na casa de um amigo e quer sair:

🌧️ Exemplo do Dia a Dia

SE estiver a chover, levo guarda-chuva.
SENÃO (se não estiver), não levo.

Na programação, usamos IF (se) e ELSE (senão) para fazer o computador tomar decisões assim. É como ensinar o computador a pensar!

✅ Condição VERDADEIRA

O código dentro do IF executa

❌ Condição FALSA

O código dentro do ELSE executa (ou nada acontece)

Sempre que você vê "SE... ENTÃO..." no português, provavelmente vai usar um IF no código!

🔵 IF Simples (Somente "SE")

O IF mais básico verifica apenas uma condição. Se for verdade, faz algo. Se for falsa, continua o programa normalmente.

📱 Exemplo: Verificar Idade para Entrar no Facebook

📝 Em Português:

SE a idade for maior ou igual a 13 anos, pode criar conta.
Caso contrário, não fazemos nada especial.

JavaScript // Guardamos a idade numa "caixa" chamada idade let idade = 15; // IF = SE if (idade >= 13) { console.log("Pode criar conta no Facebook!"); } // Esta linha SEMPRE executa, IF ou não console.log("Fim do programa");
Início
⬇️
Idade >= 13?
⬇️ SIM
Mostrar "Pode criar conta"
⬇️
Fim
Não esqueça: o sinal >= significa "maior ou igual". É diferente de > que é só "maior".

🟢 IF e ELSE (SE... SENÃO...)

Agora vamos aprender a fazer duas coisas diferentes: uma se a condição for verdadeira, outra se for falsa.

💰 Exemplo: Comprar Pão na Padaria

📝 Em Português:

SE tiver 50 MTN ou mais, compro o pão.
SENÃO, fico em casa porque não tenho dinheiro suficiente.

JavaScript let dinheiro = 30; if (dinheiro >= 50) { console.log("Compro o pão! 🍞"); } else { console.log("Não tenho dinheiro suficiente... 😢"); }
1

O computador verifica a condição

dinheiro (30) >= 50? Resultado: FALSO

2

Como é FALSO, pula o IF e vai para o ELSE

Executa o que está dentro das chaves do ELSE

3

Resultado na tela:

"Não tenho dinheiro suficiente... 😢"

Use ELSE quando quiser que o programa faça algo específico quando a condição NÃO for atendida.

🔴 Operadores de Comparação (Muito Importante!)

Para fazer condições, precisamos comparar valores. Aqui estão os operadores mais usados:

== igual a (5 == 5 → verdadeiro) != diferente de (5 != 3 → verdadeiro) > maior que (10 > 5 → verdadeiro) < menor que (3 < 7 → verdadeiro) >= maior ou igual (5 >= 5 → verdadeiro) <= menor ou igual (4 <= 5 → verdadeiro)
== (dois iguais) compara valores. = (um igual) guarda valor numa variável. Não confunda!

📋 Exemplos Práticos:

  • nota == 20 → A nota é exatamente 20?
  • senha != "1234" → A senha é diferente de "1234"?
  • saldo > 0 → O saldo é maior que zero?
  • idade >= 18 → Tem 18 anos ou mais?

📚 Mais Exemplos do Dia a Dia

🎓 Exemplo 1: Aprovação na Escola

let nota = 14; if (nota >= 10) { console.log("Aprovado! 🎉"); } else { console.log("Reprovado... Estude mais 📚"); }

🔋 Exemplo 2: Bateria do Telefone

let bateria = 15; if (bateria <= 20) { console.log("Bateria fraca! Carregue já! 🔌"); } else { console.log("Bateria OK ✅"); }

🍽️ Exemplo 3: Almoço em Casa

let hora = 13; if (hora == 13) { console.log("Hora do almoço! 🍛"); } else { console.log("Ainda não é hora de comer"); }
