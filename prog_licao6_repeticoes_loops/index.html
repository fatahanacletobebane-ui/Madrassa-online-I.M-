Repetições (Loops)

Aprenda a automatizar tarefas repetitivas com for, while e do...while. Domine o poder da iteração!

1

O que são Loops?

Imagine que você precisa imprimir "Olá, Mundo!" 100 vezes. Copiar e colar o mesmo código 100 vezes seria trabalhoso e ineficiente. É aí que entram os loops (laços de repetição) - estruturas que permitem executar um bloco de código múltiplas vezes.

Visualização: Imprimir números de 1 a 5

1
💡 Dica Importante

Loops são fundamentais em programação. Eles aparecem em praticamente todos os programas, desde simples contadores até processamento de dados complexos.

2

O Loop FOR

O for é o loop mais comum em JavaScript. Ele é ideal quando você sabe exatamente quantas vezes precisa repetir algo.

Sintaxe do FOR

JavaScript 
for (// inicialização; // condição; // incremento) {
    // código a ser executado
}

Exemplo Prático

JavaScript 
// Contar de 1 até 5
for (let i = 1; i <= 5; i++) {
    console.log("Contagem: " + i);
}

// Saída:
// Contagem: 1
// Contagem: 2
// Contagem: 3
// Contagem: 4
// Contagem: 5
🎯 Entendendo as Partes

Inicialização (let i = 1): Executada uma vez antes do loop começar.
Condição (i <= 5): Verificada antes de cada iteração. Se for falsa, o loop para.
Incremento (i++): Executado após cada iteração (i++ é equivalente a i = i + 1).

O Loop WHILE

O while é usado quando você não sabe exatamente quantas repetições serão necessárias, mas sabe a condição que deve ser atendida para continuar.

JavaScript 
let contador = 1;

while (contador <= 5) {
    console.log("Contagem: " + contador);
    contador++; // Não esqueça de incrementar!
}
⚠️ Cuidado com Loops Infinitos!

Se você esquecer de incrementar a variável no while, o loop nunca terminará e travará seu programa. Sempre certifique-se de que a condição eventualmente se tornará falsa.

Exemplo: Senha Correta

JavaScript 
let senha = "";

while (senha !== "1234") {
    senha = prompt("Digite a senha:");
    
    if (senha !== "1234") {
        alert("Senha incorreta! Tente novamente.");
    }
}

alert("Acesso permitido!");
4

O Loop DO...WHILE

O do...while é similar ao while, mas garante que o código seja executado pelo menos uma vez, pois a condição é verificada no final.

JavaScript 
let resposta;

do {
    resposta = prompt("Você quer continuar? (s/n)");
} while (resposta === "s");

console.log("Programa encerrado!");

Comparação dos Loops

Loop Quando Usar Verificação
for Número exato de repetições conhecido No início
while Condição de parada conhecida, repetições indefinidas No início
do...while Código deve executar pelo menos uma vez No final

Exemplos Práticos

1. Tabuada

JavaScript 
let numero = 7;

console.log("Tabuada do " + numero + ":");
for (let i = 1; i <= 10; i++) {
    let resultado = numero * i;
    console.log(numero + " x " + i + " = " + resultado);
}

2. Soma de Números

JavaScript 
let soma = 0;

for (let i = 1; i <= 100; i++) {
    soma = soma + i; // ou soma += i;
}

console.log("Soma de 1 a 100: " + soma); // 5050

3. Contagem Regressiva

JavaScript 
console.log("Iniciando contagem regressiva...");

for (let i = 10; i >= 0; i--) {
    console.log(i);
    
    if (i === 0) {
        console.log("🚀 Lançamento!");
    }
}

Exercícios Práticos

Fácil Exercício 1: Contar Pares

Crie um loop que imprima todos os números pares de 2 a 20.

JavaScript 
// Escreva seu código aqui
for (/* complete */) {
    /* complete */
}
✅ Correto! Você pode usar: for (let i = 2; i <= 20; i += 2) ou verificar if (i % 2 === 0)
Médio Exercício 2: Fatorial

Calcule o fatorial de 5 (5! = 5 × 4 × 3 × 2 × 1 = 120) usando um loop.

JavaScript 
let numero = 5;
let fatorial = 1;

// Complete o loop
✅ Resposta: for (let i = 1; i <= numero; i++) { fatorial *= i; }
