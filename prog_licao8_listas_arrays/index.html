Arrays (Listas)

Aprenda a armazenar múltiplos valores em uma única variável e manipular coleções de dados!

1

O que são Arrays?

Um Array é uma estrutura de dados que armazena múltiplos valores em uma única variável. Pense nele como uma lista ordenada de itens.

Visualização: Array de Frutas
🍎[0]
🍌[1]
🍇[2]
🍊[3]
💡 Índices Importantes

Arrays em JavaScript começam no índice 0 (zero). O primeiro elemento está na posição 0, o segundo na 1, e assim por diante!

2

Criando Arrays

Formas de Criar

JavaScript 
// Usando colchetes (mais comum)
let frutas = ["Maçã", "Banana", "Laranja"];

// Array vazio
let numeros = [];

// Com new Array()
let carros = new Array("Fusca", "Gol", "Civic");

Acessando Elementos

JavaScript 
console.log(frutas[0]);    // "Maçã"
console.log(frutas[1]);    // "Banana"
console.log(frutas[2]);    // "Laranja"

// Último elemento
let ultimo = frutas[frutas.length - 1];
🎯 Dica Pro

Use frutas.at(-1) para pegar o último elemento sem calcular o length!

3

Modificando Arrays

Principais Métodos

📥 push()
Adiciona um elemento ao FINAL do array
📤 pop()
Remove o ÚLTIMO elemento e retorna ele
📤 shift()
Remove o PRIMEIRO elemento e retorna ele
📥 unshift()
Adiciona um elemento no INÍCIO do array
✂️ splice()
Remove/Adiciona elementos em qualquer posição
🔍 indexOf()
Encontra a posição de um elemento
JavaScript 
let cores = ["Vermelho", "Verde", "Azul"];

cores.push("Amarelo");        // ["Vermelho", "Verde", "Azul", "Amarelo"]
cores.pop();                  // Remove "Amarelo"
cores.unshift("Preto");     // ["Preto", "Vermelho", "Verde", "Azul"]
cores.shift();                // Remove "Preto"

// splice(posição, quantosRemover, adicionar...)
cores.splice(1, 1, "Rosa"); // ["Vermelho", "Rosa", "Azul"]
4

Percorrendo Arrays

Combinando Arrays com Loops, podemos processar cada elemento:

JavaScript 
let notas = [8.5, 9.0, 7.5, 10];

for (let i = 0; i < notas.length; i++) {
    console.log("Nota " + i + ": " + notas[i]);
}

// Somando todas as notas
let soma = 0;
for (let nota of notas) {
    soma += nota;
}
console.log("Média: " + soma / notas.length);

for...of vs for...in

Loop Uso Retorna
for...of Valores do array Elemento
For...in Índices do array Índice (0, 1, 2...)
5

Métodos Avançados

map() - Transforma cada elemento

JavaScript 
let numeros = [1, 2, 3, 4];

let dobrados = numeros.map(num => num * 2);
// [2, 4, 6, 8]

filter() - Filtra elementos

JavaScript 
let idades = [15, 22, 18, 12, 30];

let maiores = idades.filter(idade => idade >= 18);
// [22, 18, 30]

reduce() - Reduz a um valor

JavaScript 
let valores = [10, 20, 30];

let total = valores.reduce((acc, val) => acc + val, 0);
// 60 (10+20+30)
🚀 Métodos Modernos

Esses métodos (map, filter, reduce) são extremamente poderosos e são usados constantemente no desenvolvimento profissional!

Exemplos Práticos

1. Lista de Compras

JavaScript 
let compras = ["Leite", "Pão", "Ovos"];

// Adicionar item
compras.push("Manteiga");

// Marcar como comprado (remover)
let item = prompt("O que você comprou?");
let posicao = compras.indexOf(item);

if (posicao !== -1) {
    compras.splice(posicao, 1);
    console.log(item + " comprado! ✅");
} else {
    console.log("Item não encontrado na lista");
}

2. Maior e Menor Valor

JavaScript 
let valores = [45, 12, 89, 33, 5];

let maior = Math.max(...valores);
let menor = Math.min(...valores);

console.log("Maior: " + maior);   // 89
console.log("Menor: " + menor);   // 5

3. Array de Objetos

JavaScript 
let alunos = [
    { nome: "Ana", nota: 8.5 },
    { nome: "Bruno", nota: 7.0 },
    { nome: "Carlos", nota: 9.5 }
];

// Encontrar alunos aprovados (nota >= 7)
let aprovados = alunos.filter(aluno => aluno.nota >= 7);

aprovados.forEach(aluno => {
    console.log(aluno.nome + " ✅");
});

Exercícios Práticos

Fácil Exercício 1: Soma do Array

Crie uma função que recebe um array de números e retorna a soma de todos os elementos. Use o método reduce.

JavaScript 
function somarArray(numeros) {
    // Use reduce aqui
}
✅ Resposta: return numeros.reduce((a, b) => a + b, 0);
Médio Exercício 2: Remover Duplicados

Crie uma função que remove valores duplicados de um array. Dica: Use filter com indexOf.

💡 Dica: return array.filter((item, index) => array.indexOf(item) === index);
Desafio Exercício 3: Ordenar Array

Crie uma função que ordena um array de números em ordem crescente sem usar o método sort(). Implemente o algoritmo Bubble Sort!

🧠 Algoritmo Bubble Sort:
for (let i = 0; i < array.length; i++) {
  for (let j = 0; j < array.length - 1; j++) {
    if (array[j] > array[j + 1]) {
      [array[j], array[j + 1]] = [array[j + 1], array[j]];
    }
  }
}
